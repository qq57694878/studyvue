1.安装nodejs
  https://npm.taobao.org/mirrors/node/v8.9.3/node-v8.9.3-x64.msi
  如出现2503,2502错误
  msiexec /package node-v8.9.3-x64.msi
2.安装vue-cli
  npm install --global vue-cli
3.用vue-cli 构建一个vue项目;cmd:
  vue init webpack vue-node
  cd vue-node
  npm install
  npm run dev
